Linux Lopputyö 2020

Pinta is an open-source, cross-platform bitmap image drawing and editing program inspired by Paint.NET, a similar image editing program which is limited to Microsoft Windows. Pinta has more features than Microsoft Paint. Compared with open-source image editor GIMP, Pinta is simpler and has fewer features.


1. Aloita klikkaamalla ruudun vasemmasta alanurkasta "Show applications", ja kirjoita hakuun "gimp". Listasta tulisi löytyä GNU Image Manipulation Program. Valitse ohjelma listasta.

2. Tämän jälkeen aukeaa lista josta valitaan GNU Image Manipulation Program.
3. Klikkaa "Install", ja asennuksen tulisi kestää korkeintaan muutaman minuuutin.
4. Tämän jälkeen sinulta kysytään käyttäjäsi salasanaa, jotta asennus voidaan vahvistaa.
5. Kun ohjelma on asennettu, voit käynnistää sen klikkaamalla "Launch".
6. Ohjelman aukemiseen menee muutama minuutti.
